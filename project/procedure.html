Example

Que. Find the GCD of 270 and 192

Solution:

Step 1:

a=270,b=192

a ≠ 0

b ≠ 0

Use long division to find that 270/192=1 with a remainder of 78.

We can write this as:

270=192*1+78 Find GCD(192,78),since GCD(270,192)=GCD(192,78)

Step 2:

a=192,b=78

a ≠ 0

b ≠ 0

Use long division to find that 192/78=2 with a remainder of 36.

We can write this as:

192=78*2+36

Find GCD(78,36),since GCD(192,78)=GCD(78,36)

Step 3:

a=78,b=36

a ≠ 0

b ≠ 0

Use long division to find that 78/36=2 with remainder of 6.

We can write this as:

78=36*2+6

Find GCD(36,6),since GCD(78,36)=GCD(36,6)

Step 4:

a=36,b=6

a ≠ 0

b ≠ 0

Use long division to find that 36/6=6 with a remainder of 0.

We can write this as:

36=6*6+0

Find GCD(6,0),since GCD(36,6)=GCD(6,0)

Step 5:

a=6,b=0

a ≠ 0

b=0,GCD(6,0)=6

So we have shown:

GCD(270,192)= GCD(192,78)= GCD(78,36)= GCD(36,6)=GCD(6,0)=6

GCD(270,192)=6
